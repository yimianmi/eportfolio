Designing Integrated Media Environments (EME 6208)

About This Portfolio

Welcome to my e-Portfolio for EME 6208: Designing Integrated Media Environments. Throughout this course, I explored multimedia design through image editing, web design, video production, digital animation, screencasting, and AI-supported media creation.

As a composer and educator, I learned how multimedia tools can support teaching and communication in creative ways. This portfolio brings together all the skills I practiced throughout the semester.

Image Editing

Bitmap and Vector Imagery

For this project, I created two pieces: a book cover design and a personal logo. I used layering, masking, color blending, and balance to create expressive visuals using both bitmap and vector techniques.

This project helped me understand how composition, color harmony, and image resolution influence clarity in both print and digital formats.

Web Design

Interactive Quiz & Extended Chords Quiz

I created two web-based learning tools: a 10-question music theory quiz and an extended chords quiz.
